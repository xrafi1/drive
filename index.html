Public Photo Uploader

Snap a photo, upload it to a specific Google Drive folder, and get a public link.

Setup Required

To use this tool, you need your own Google API Credentials and a destination Folder ID.

  1. Go to the Google Cloud Console.
  2. Create a new project and enable the "Google Drive API".
  3. Go to "APIs & Services" > "Credentials".
  4. Click "+ CREATE CREDENTIALS" and create an "API key". This is for the `API_KEY` variable. **Do not use a Client Secret.**
  5. Click "+ CREATE CREDENTIALS" again and create an "OAuth 2.0 Client ID".
  6. Choose "Web application". Under "Authorized JavaScript origins", add your GitHub Pages URL (e.g., `https://your-username.github.io`).
  7. Copy the "Client ID". Paste both new credentials in the script below.
  8. Navigate to the Google Drive folder you want to upload to and copy its ID from the URL.
  9. Paste the Folder ID into the `FOLDER_ID` variable in the script below.
